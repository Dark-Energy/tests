Глубокое погружение во фракталы ограничено пределом физической точности вычислений для чисел с плавающей точкой - не больше 15 значимых цифр. Решить проблему возможно с помощью алгоритмов вычислений с произвольной точностью. Для javascript существует несколько библиотек - Big, Bignumber, math, decimal. Все они написаны одним автором и, причём последняя даже оптимизирована под asm.js. Вряд ли существует что-то более быстрое, если здесь уместно слово "быстрое". Однако результат всё равно оказался лучше, чем Питон. Для большей скорости можно использовать возможности графического процессора (CUDE, OpenCL, шейдеры), либо использовать иные системы, чем настольный компьютер.
