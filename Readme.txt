Наброски и прототипы.


Fractals - отрисовка фрактала burning ship в браузере, получилось не совсем то, чего хотелось бы. Браузер неподходящая платформа для таких вычислений.

Dynalinks - каталог ссылок с категориями и тэгами, который я сделал для своих личных нужд. Мне хотелось, чтобы приложение работало без расширений в бразуере и без сервера. Поэтому сделано без затей. База данных хранится в обычном файле в формате json и загружается с помощью элемента script. Сначала я вообще хотел сделать без лишних зависимостей, поэтому там нет никакого jquery. Но потом решил, что это слишком муторно. В качестве фреймовка используется чистый knockout. Angular показался мне громоздким и тормознутым. Получившийся результат меня удовлетворил, но в следующий раз я всё-таки использую сервер с реляционной базой данных.
